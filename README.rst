
RPI Fan Control
===============

Raspberry Pi fan controller is Python script to start and stop  5V fan depends on cpu temperature.

Software requirements
=====================

To be able to run **RPI Fan Control** you have to minimum meet following requirements:

- gpiozero==1.5.1
- RPi.GPIO==0.7.0
